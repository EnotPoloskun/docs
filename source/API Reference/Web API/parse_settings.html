get

Pull down the settings already configured for parsing incoming email.

  • JSON
  • XML

Call

{% codeblock %} https://sendgrid.com/api/parse.get.json?api_user=youremail@domain.com&api_key=secureSecret {% endcodeblock %}

Response: Success

{% codeblock lang:javascript %} {"parse":[{"hostname":"www.example.com","url":"www.mydomain.com\/parse.php","spam_check":1}]} {% endcodeblock %}

Response: Empty

{% codeblock lang:javascript %} {"parse":[]} {% endcodeblock %}

Call

{% codeblock %} https://sendgrid.com/api/parse.get.xml?api_user=youremail@domain.com&api_key=secureSecret {% endcodeblock %}

Response: Success

{% codeblock lang:xml %} www.example.com www.mydomain.com/parse.php 1 {% endcodeblock %}

Response: Empty

{% codeblock %} {% endcodeblock %}
