This module allows you to add new Variations to your drafted Newsletters. These Variations can be used in A/B testing.

add

Create a new Variation.

Parameters

Parameter Required Requirements Description
name Yes Must be an existing Newsletter in draft mode. The name of the Newsletter you are adding Variations to.
variation Yes None The name of the Variation you will be adding.
subject Yes None The subject that will be used for the Variation being created.
text Yes None The text portion of the Newsletter Variation being added.
html Yes None The html portion of the Newsletter Variation being added.
type Yes text/html The Variation type.
identity_name No Must be an existing Newsletter Identity Identity name that is needed if the Newsletter name does not already exist. Will be used to create a parent Newsletter.

edit

Edit an existing Variation.

Parameters

Parameter Required Requirements Description
name Yes Must be an existing Newsletter with Variations added to it. The name of the Newsletter you are editing Variations on.
variation Yes Must be an existing Variation The name of the Variation you will be editing.
newvariation No None The new name of the Variation.
subject No None The new subject of the Variation.
text No None The new text portion of the Newsletter Variation.
html No None The new html portion of the Newsletter Variation.
type No text/html The Variation type.

get

Retrieve the contents of an existing Variation.

Parameters

Parameter Required Requirements Description
name Yes Must be an existing Newsletter The name of the Newsletter with existing Variations added to it.
variation Yes Must be an existing Variation The name of the Variation you would like the contents of.

cancel

Cancels a current Variation send that is in progress.

Parameters

Parameter Required Requirements Description
name Yes Must be an existing Newsletter The name of the Newsletter you wish to cancel, this cancels added Variations.

delete

Delete the contents of an existing Variation.

Parameters

Parameter Required Requirements Description
name Yes Must be an existing Newsletter The name of the Newsletter with existing Variations added to it.
variation No Must be an existing Variation The name of the Variation you would like to delete.

list

Retrieve a list of all Variations attached to a Newsletter.

Parameters

Parameter Required Requirements Description
name Yes Must be an existing Newsletter Select the Newsletter you'de like to list the Variations of.
variation No Must be an existing Variation An optional list of names's to check for.

pickwinner

Select the Variation you would like to send.

Parameters

Parameter Required Requirements Description
name Yes Must be an existing Newsletter Select the Newsletter you're picking the winner of.
variation Yes Must be an existing Variation The winning Variation.

schedule

Schedule a send time and percentage for your variations.

Parameters

Parameter Required Requirements Description
name Yes Must be an existing Newsletter Select the Newsletter you'de like to schedule Variation testing on.
percent Yes Must be a number 0-100 Percentage of the total recipients that will be emailed in the test.
delay No Must be an existing Variation An optional list of names's to check for.
delay_type No Must be an existing Variation An optional list of names's to check for.
date No Must be an existing Variation An optional list of names's to check for.
timezone No Must be an existing Variation An optional timezone of specific date (timezones are in the Olson Database name format).
