--- layout: page weight: 0 title: Filter Commands navigation: show: true --- {% anchor h2 %} Get Available {% endanchor %}

List all of the available apps.

Note: The name entry is used in all the other API calls to identify a app. For a list of all the apps and parameters accesible with the Web API you can check the Filter Settings page.

{% xmljsontabs getavailable %}

Call

{% requestblock %} {% requesturl GET %}https://sendgrid.com/api/filter.getavailable.json?api_user=your_sendgrid_username&api_key=your_sendgrid_password{% endrequesturl %} {% endrequestblock %}

Response

{% codeblock lang:json %}[ ... { "name": "bcc", "title": "BCC", "description": "Automatically BCC an address for every e-mail sent.", "activated": false }, ... ] {% endcodeblock %}

Call

{% requestblock %} {% requesturl GET %}https://sendgrid.com/api/filter.getavailable.xml?api_user=your_sendgrid_username&api_key=your_sendgrid_password{% endrequesturl %} {% endrequestblock %}

Response

{% codeblock lang:xml %} ... bcc BCC Automatically BCC an address for every e-mail sent. 0 ... {% endcodeblock %}

{% anchor h2 %} Get App Settings {% endanchor %}

Get the Settings Of An App

{% xmljsontabs getsettings %}

Call

{% requestblock %} {% requesturl GET %}https://sendgrid.com/api/filter.getsettings.json?api_user=your_sendgrid_username&api_key=your_sendgrid_password&name=bcc{% endrequesturl %} {% endrequestblock %}

Response: Success

{% codeblock lang:json %} {"message":"success","settings":[{"bcc":"nick@sendgrid.com"}]} {% endcodeblock %}

Response: Error

{% codeblock lang:json %}{ "message": "error", "errors": [ "...error messages..." ] } {% endcodeblock %}

Response: Empty

*This is returned if that filter has no settings or is not enabled. {% codeblock lang:json %}{ "settings": null } {% endcodeblock %}

Call

{% requestblock %} {% requesturl GET %}https://sendgrid.com/api/filter.getsettings.xml?api_user=your_sendgrid_username&api_key=your_sendgrid_password&name=bcc{% endrequesturl %} {% endrequestblock %}

Response: Success

{% codeblock lang:xml %} nick@sendgrid.com {% endcodeblock %}

Response: Error

{% codeblock lang:xml %} error ... error messages ... {% endcodeblock %}

Response: Empty

*This is returned if that filter has no settings or is not enabled. {% codeblock lang:xml %} {% endcodeblock %}
