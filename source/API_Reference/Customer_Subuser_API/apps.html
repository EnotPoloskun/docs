List

List all of the available apps. Note: The name entry is used in all the other API calls to identify a app.
Parameter Required Requirements Description
task Yes Must be set to getavailable Retrieve Available Apps
user Yes Subuser must be registered under your account The subuser who we will update
  • JSON
  • XML

Call

{% codeblock %} https://sendgrid.com/apiv2/customer.apps.json?api_user=username&api_key=secureSecret&task=getavailable&user=example@example.com {% endcodeblock %}

Response: Success

{% codeblock lang:javascript %} {"name":"twitter","title":"Twitter","description":"This plugin allows you to send an email message to twitter","activated":false}, ... {% endcodeblock %}

Call

{% codeblock %} https://sendgrid.com/apiv2/customer.apps.xml?api_user=username&api_key=secureSecret&task=getavailable&user=example@example.com {% endcodeblock %}

Response: Success

{% codeblock lang:xml%} twitter Twitter This plugin allows you to send an email message to twitter 0 {% endcodeblock %}

Customize App

These API calls require that settings are passed using POST.
  • JSON
  • XML

Call

{% codeblock %} https://sendgrid.com/apiv2/customer.apps.json?api_user=username&api_key=secureSecret&name=twitter&task=setup&user=example@example.com {% endcodeblock %}

Response: Success

{% codeblock lang:javascript%} "message":"success","settings":{"field_name":"field_value”} {% endcodeblock %}

Return - Empty

This is returned if that filter has no settings or is not enabled. {% codeblock lang:javascript%} "settings":null {% endcodeblock %}

Call

{% codeblock %} https://sendgrid.com/apiv2/customer.apps.xml?api_user=username&api_key=secureSecret&name=twitter&task=setup&user=example@example.com {% endcodeblock %}

Response: Success

{% codeblock lang:xml %} field_value ... {% endcodeblock %}

Response: Error
