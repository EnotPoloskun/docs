--- layout: page weight: 0 title: Node.js navigation: show: true --- To send email over SMTP using Node.js the node_mailer library is a good option. To install node_mailer, use npm. {% codeblock %} $ npm install mailer {% endcodeblock %} Then create a file called mail.js: {% codeblock lang:javascript %} var email = require("mailer"); var sgusername = 'sendgrid_username'; var sgpassword = 'sendgrid_password'; email.send({ host : "smtp.sendgrid.net", port : "587", domain : "yourdomain.com", to : "recipient@domain.com", from : "yourname@yourdomain.com", subject : "This is a subject", body: "Hello, this is a test body", authentication : "login", username : sgusername, password : sgpassword }, function(err, result){ if(err){ console.log(err); } }); {% endcodeblock %} To execute mail.js run the following {% codeblock %} $ node mail.js {% endcodeblock %}
