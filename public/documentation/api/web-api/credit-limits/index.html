Parameter Required Description

userid

yes

Command

https://sendgrid.com//api/credits/getlimit.xml?userid=id&api;_user=youremail@domain.com&api;_key=secureSecret

Return

`

200 175 daily 2011-02-21

`

Return - Error

`

 error 
...

`

JSON

Command

` { “success”: {

  "credits_allowed":"200",

} } `
